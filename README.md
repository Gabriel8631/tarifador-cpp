#  Tarifador em C++

Este é um projeto simples de um sistema de bilhetagem/tarifação criado em C++. 

## O que ele faz?
O programa recebe a duração de uma chamada telefônica em minutos e calcula o valor final da conta com base em uma tarifa pré-estabelecida (R$ 0,50 por minuto).

## Tecnologias
* C++
* Bibliotecas padrão (`iostream`, `iomanip`)
